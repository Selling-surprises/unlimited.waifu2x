## Unlimited Waifu2x 使用说明

### 📌 网站简介

**Unlimited Waifu2x是一款基于深度卷积神经网络（CNN）的免费在线图片无损放大工具，专门针对动漫风格图像优化，同时也支持普通照片处理。**

**当前网页复刻了（unlimited.waifu2x.net）并精简了一些模型。**

与原版 Waifu2x（waifu2x.udp.net）相比，**Unlimited 版本的主要优势**：
- ✅ **无文件大小限制**（原版限制5MB）
- ✅ **支持 4 倍放大**（原版在线版仅支持2倍）
- ✅ **本地浏览器处理**，不上传服务器，隐私更安全

---

### 🎯 核心功能

| 功能 | 说明 |
|------|------|
| **AI 无损放大** | 支持 1.6x / 2x / 4x 放大倍数 |
| **智能降噪** | 两档降噪：高、最高 |
| **风格识别** | 自动优化动漫/插画线条，支持照片模式 |
| **格式输出** | PNG / WebP 格式可选 |
| **本地处理** | 浏览器内完成，保护隐私 |

---

### 📝 使用步骤

1. **访问网站**  

   打开 https://selling-surprises.github.io/Unlimited-Waifu2x/

3. **上传图片**  

   - 支持拖拽上传（Drag & Drop）
   - 或点击"选择文件"从本地上传
   - 也支持输入在线图片 URL

5. **选择参数**  

   - **风格**：插图 / 照片（动漫选"插图"）
   - **降噪程度**：仅在图片有噪点时使用，否则可能产生反效果
   - **放大倍数**：2x / 4x
   - **输出格式**：PNG 或 WebP

7. **开始处理**  

   点击"转换"按钮，等待 AI 处理完成

9. **下载结果**  

   处理完成后点击"下载"保存高清图片

---

### ⚠️ 重要提示

**尺寸限制**（原版限制，Unlimited版本理论上无限制但仍需注意浏览器性能）：
- 可降噪图像最大尺寸：3000×3000px
- 可放大图像最大尺寸：1500×1500px（建议）

**使用建议**：

- 🎨 **动漫/插画**：效果最佳，线条锐利、色块纯净
- 📸 **照片**：选择"照片"模式，但效果不如专用照片放大工具
- 🔇 **降噪慎用**：仅在图片本身有噪点时使用，无噪点图片降噪会产生模糊
- 🖥️ **性能要求**：本地处理依赖设备性能，大图片处理可能需要较长时间

### 🔒 隐私说明

Unlimited Waifu2x 采用**浏览器本地处理**技术，图片数据不会上传到远程服务器，适合处理包含隐私内容的图片。

---

### 📚 技术背景

Waifu2x 由日本开发者 nagadomi 开源，基于**深度卷积神经网络**技术，通过学习大量高清动漫图像建立低分辨率到高分辨率的映射模型。相比传统插值算法，能更好还原纹理细节，避免线条锯齿和色块模糊。

**开源地址**：https://github.com/nagadomi/nunif
